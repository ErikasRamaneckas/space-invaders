# Space Invaders

The classic arcade game "Space Invaders" built with TypeScript and HTML Canvas.

Control your spaceship and kill all alien invaders without losing all of your lives.

**Demo link**: [Space Invaders](https://erikasramaneckas.github.io/space-invaders/)

<img src="assets/images/screenshot.png" width="300" height="300" alt="screenshot of Space Invaders gameplay" />

## How to Play

- Left Arrow Key - move left
- Right Arrow Key - move right
- Spacebar - shoot lasers
